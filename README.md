# 本地卡片学习（增强版）

一个功能强大的本地化卡片学习工具，支持智能导入、语音朗读、多组管理和详细学习统计。

### 🎯 智能导入系统

- **多格式支持**：智能识别 Q:/A:、数字题号、答：前缀等多种格式
- **文档导入**：支持 .docx 文件导入（使用 mammoth 库）
- **JSON 导入导出**：完整的数据备份和迁移功能
- **实时预览**：导入前可预览和编辑所有题目

### 🎨 智能主题系统

- **三模式切换**：自动模式（根据时间切换）、手动亮色、手动暗色
- **时间感知**：晚上 18:00 至早上 6:00 自动使用暗色主题
- **系统适配**：可跟随系统主题偏好

### 🔊 多媒体学习体验

- **语音朗读 (TTS)**：支持题目和答案朗读
- **独立控制**：可分别控制题目和答案的朗读
- **音效反馈**：正确/错误提示音和倒计时警告音
- **音量调节**：独立的 TTS 音量控制

### 📚 学习模式

- **学习模式**：所有题目过筛，确保全面掌握
- **复习模式**：专注于不熟和不会的题目
- **智能队列**：自动管理学习进度和复习安排

### 🗂️ 多组管理

- **独立存储**：每组数据保存在 localStorage 中互不干扰
- **灵活管理**：支持创建、重命名、删除组
- **批量操作**：支持多选、批量删除、重置统计

## 📊 学习统计

### 详细数据追踪

- **掌握程度**：会、不熟、不会的详细计数
- **进度监控**：实时显示学习进度和队列状态
- **CSV 报表**：支持导出详细学习报告
- **可视化统计**：表格形式展示每道题的学习情况

### 智能学习算法

- **掌握判断**：正确≥2次且无错误自动标记为掌握
- **混合统计**：既有正确又有错误自动标记为不熟
- **复习优化**：自动将不熟悉的题目加入复习队列

## ⚙️ 自定义设置

### 学习参数

- **倒计时**：可调节题目显示时间（默认15秒）
- **警告提醒**：自定义最后提醒秒数（默认5秒）
- **错误延迟**：答错后的显示延迟时间（默认3秒）

### 界面控制

- **响应式设计**：完美适配桌面和移动设备
- **紧凑布局**：优化的小屏幕显示效果
- **直观操作**：清晰的按钮状态和提示信息

## 🚀 快速开始

1. **创建题目组**：点击"新建组"创建学习主题
2. **导入题目**：
   - 直接粘贴文本（支持多种格式）
   - 导入 .docx 文档
   - 手动在预览界面添加
3. **开始学习**：选择模式后点击"开始学习"
4. **查看进度**：实时查看学习统计和掌握情况

## 💾 数据安全

- **纯本地存储**：所有数据保存在浏览器本地
- **数据导出**：支持 JSON 格式完整备份
- **跨组隔离**：不同组的数据完全独立
- **无云同步**：保护隐私，无需网络连接

## 🛠️ 技术特点

- **纯前端实现**：基于 HTML5、CSS3、JavaScript
- **现代 API**：使用 Web Speech API、Web Audio API
- **响应式设计**：适配各种屏幕尺寸
- **无依赖**：除 mammoth（docx 解析）外无其他依赖

## 📱 移动端优化

- **触摸友好**：大按钮设计，便于触屏操作
- **自适应布局**：表格和控件在小屏幕上自动调整
- **性能优化**：流畅的动画和响应体验

这个工具特别适合需要记忆性学习的内容，如外语单词、专业知识、考试复习等场景，通过科学的间隔重复算法和多媒体辅助，大幅提升学习效率。
